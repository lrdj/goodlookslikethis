Report an issue with the website USA.gov

Use this form to report broken links or errors you experienced while using USA.gov.

This form is only for reporting problems with the content or function of USA.gov.

  • We are not able to help with personal issues. 
    • We cannot respond to or forward requests for personal help.
  • We cannot fix problems on other websites we link to.

Please let us know about issues you find on the USA.gov website including:

  • Hyperlinks that are broken or go to the wrong place
  • Mistakes in our content
  • Out-of-date phone numbers, addresses, or agency program names
  • Pages or features on our website that do not display or work properly

Please include the URL or the title of the page with the problem. 

We ask for your first name and email address in case we need to get in touch with follow-up questions. We will not use your contact information for any other purpose.

Thank you for helping us improve USA.gov.
